## Aviso

Os nomes e logótipos **Vercel**, **Next.js**, **Turbo**, **v0**, **AI SDK**, etc. são marcas registadas. Este ficheiro apenas **lista links oficiais**; cumpre os [termos](https://vercel.com/legal/terms) e as regras da página Brands antes de usar assets em produto ou marketing.

## Símbolo em texto

O símbolo Vercel pode ser representado pelo carácter Unicode **▲** (U+25B2) quando só cabe texto, segundo as guidelines oficiais.

## Código React (documentação oficial)

Exemplos típicos citados no site (requerem pacote e permissão de marca):

```tsx
import { VercelWordmark, VercelMark } from "geist/logos";
// <VercelWordmark height={32} />
// <VercelMark size={16} />
```

**Nota:** a versão `geist` no npm focada em fontes pode não expor `geist/logos` em todas as versões; confirmar no teu `node_modules` ou usar os ZIPs de imprensa.
