Float Dash (Dock Menu)

Navegação global fixa na lateral direita, comportando-se como um Dock de macOS. Foco em economia espacial e interações por proximidade do cursor.

UX Writing & Identidade

Voz: Utilitária e silenciosa.

Tom: Neutro. Este componente existe para acelerar fluxos, não para conversar.

Regra de Ouro: Os "Tooltips" (`data-float-tip`) no estado retraído não devem terminar com ponto final, devem ter 1 a 2 palavras no máximo.

Matriz FEER (Technical Rigor)

Fluxos (Flows)

Passo Ação do Usuário Resposta do Sistema
1 Aproxima o cursor do menu (150px de distância) O menu aumenta ligeiramente a opacidade e a escala num efeito de magnetismo leve (Proximity JS).
2 Pausa (hover) sobre um ícone Tooltip nativo CSS aparece flutuando à esquerda do ícone.
3 Clica no botão "Spark" #float-dash-open Alterna data-expanded="true", ocultando a pilha de ícones e revelando a navegação textual com animação CSS *clip-path* ou opacidade.

Estados de UI (States)

Estado Descrição Visual
Collapsed (data-expanded="false") Apenas a coluna de ícones visível. Fundo transparente que se materializa com hover.
Expanded (data-expanded="true") Painel estendido exibindo listas verticais (`.float-dash__row`). Superfície escura texturada ou de fundo absoluto.
Item Ativo (.is-active) Ícone fica branco puro var(--text) com contorno de acentuação leve à direita, oposto ao padrão de sidebar tradicional.

Acessibilidade (POUR)

Princípio Aplicação (HTML/CSS)
Compreensível Ícones ambíguos usam .visually-hidden para screen readers e data-float-tip para leitura visual rápida.
Robusto A animação de proximidade deteta pointer: fine e prefers-reduced-motion: no-preference. Se falhar, CSS fallback mantém operação estática e segura.
